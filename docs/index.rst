Schnellübersicht
----------------

Support-Engine ist eine moderne, mandantenfähige Helpdesk-Lösung für deutschsprachige Unternehmen.

**Kernfunktionen:**

* Multi-Tenant-Architektur mit vollständiger Datenisolierung
* Ticketverwaltung mit SLA-Tracking
* Microsoft Exchange Online Integration
* Zeiterfassung und Arbeitszeitverfolgung
* CRM mit Kunden, Kontakten und Assets
* Wissensdatenbank mit Rich-Text-Editor
* REST-API für Web und Mobile
